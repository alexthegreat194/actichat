About

Actichat is a messaging platform built for those who desire fast and simple communication at anytime. The application optimizes messaging by running in any browser or web environment, simplifying the process. In order to use the platform, the user shares an invite link or code with the appropriate audience. The recipient joins the chat using the information and is then able to send and receive messages through an interactive user interface.
